# TypeNow

A minimalist typing speed test application built with React and Tailwind CSS.

## Features

- Real-time typing speed (WPM) and accuracy tracking
- 30s, 60s, and 120s test durations
- Three-section typing display with current line highlighting
- Performance analytics with charts (powered by Recharts)
- Session history with past results
- Dark mode support with system theme detection
- Mobile responsive design
- Local storage for saving test results

## Tech Stack

- React 19
- Vite
- Tailwind CSS 4
- Radix UI (Select, Toast, Dropdown Menu)
- Recharts
- Lucide React icons
- next-themes

## Getting Started

```bash
npm install
npm run dev
```
